# chasqui.chat

Landing page for [Chasqui](https://github.com/chasqui-stack/chasqui) — built
with [Astro](https://astro.build), deployed to GitHub Pages on every push to
`main` (`.github/workflows/deploy.yml`).

```bash
nvm use          # Node 22
npm install
npm run dev      # http://localhost:4321
npm run build    # static output in dist/
```

Brand assets are copies from the parent repo
(`chasqui/docs/assets/brand/`) — that folder is the source of truth.
